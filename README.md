# Mini Quiz
Aplikacja mobilna na Androida napisana w języku Java.  
Pozwala użytkownikowi rozwiązać prosty quiz wielokrotnego wyboru.

##  Funkcje
- rozpoczęcie quizu
- losowanie 5 pytań
- wybór odpowiedzi (A, B, C)
- liczenie punktów
- wyświetlenie wyniku końcowego
- reset quizu

##  Technologie
- Android Studio
- Java
- XML (interfejs użytkownika)

## Uruchomienie
1. Pobierz projekt z repozytorium
2. Otwórz w Android Studio
3. Uruchom emulator
4. Kliknij ▶ Run


##  Struktura projektu
- `MainActivity.java` – logika aplikacji
- `Question.java` – model pytania
- `activity_main.xml` – interfejs użytkownika
